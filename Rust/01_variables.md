| 키워드 | 설명 |
| --- | --- |
| `i32` | 32비트 정수 |
| `i64` | 64비트 정수 |
| `u32` | 부호 없는 32비트 정수(unsigned int, 32bits) |
| `u64` | 부호 없는 64비트 정수(unsigned int, 64bits) |
| `isize` | 시스템 포인터 크기 정수 |
| `usize` | 시스템 포인터 크기 부호 없는 정수 |

<br>

| 키워드 | Rust에서 사용 가능 여부 | 용도 |
| --- | --- | --- |
| `int`  | ❌ 불가능 | 대신 `i32` / `u32` 등 사용 |
| `let` | ⭕ 가능 | 지역 변수 선언 |
| `let mut` | ⭕ 가능 | 변경 가능한 지역 변수 |
| `const` | ⭕ 가능 | 전역/지역 불변 상수 |
| `static` | ⭕ 가능 | 전역 정적 변수(unsafe 가능) |
| `var`  | ❌ 불가능 | 지원 안 함 |
